# 비행기 게임 개선

* 게임 설명
  -
  - 비행기를 조작하여 적을 맞추는 게임
  - 적을 5번 맞추면 적 파괴
  - 마우스 위치 이동으로 비행기 조작
  - 3D이지만, 카메라 시점이 Top view 방식
* 보완 및 업그레이드
  - 
  - 잘못된 수치 수정
  - 배경 이미지 추가
  - 피격음 추가
  - 스코어 추가
  - 각각 랜덤한 시기에 적 생성
  - 적이 공격하는 기능 추가
  - 피격 시 카메라 흔들림 기능 추가
  - 운석 추가
  - 로비, 게임오버 화면(Scene) 추가
